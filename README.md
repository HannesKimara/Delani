# Delani Studio
This is an portfolio website for Delani Studio.
### Getting Started
- Clone this repository:
    ```
    $ git clone https://github.com/HannesKimara/Delani.git
    ```
- cd into `Delani`:

    ```
    $ cd Delani 
    ```
### Usage
- Open `index.html`
- You could also visit the live demo [here](https://hanneskimara.github.io/Delani/)

### Built with
- HTML
- CSS
- JavaScript
- JQuery
- Bootstrap
